# ■ WebBasic
### 3. Tag
1. 웹페이지의 구성요소를 알려주는 표시언어
2. h1~h6 제목
   - h1 : 로고, 상호명
   - h2 : 주메뉴
   - h3~6 중요도 상세메뉴 (소제목)
3. p, pre : 문단
   - p : 일반 문단
   - pre : 줄바꿈과 공백을 유지하는 서식 텍스트
4. img : 이미지
   - src : 경로, alt : 대체용어(접근성)
5. a : 링크
   - href : 다른페이나 파일로 연결. target="_black"
   새창열기
6. 리스트태그 : ul, ol, dl
   - ul : 순서 없는 리스트
   - ol : 순서 있는 리스트
   - dl : 설명 리스트
7. 그룹핑 : div
