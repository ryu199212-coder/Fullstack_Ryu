✅ 해결 방법
원격 변경사항을 로컬로 가져오기

bash
git pull origin master
이 명령어는 원격 저장소의 master 브랜치 내용을 로컬 브랜치에 병합합니다.

병합 충돌이 없거나 해결했다면 다시 푸시

bash
git push origin master
